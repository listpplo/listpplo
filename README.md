## 👨‍💻 About Me

I am an **Automation Engineer and Software Developer** focused on building real-world industrial applications for manufacturing automation.

My work mainly includes:

- PLC communication and machine interlocking
- Traceability systems for production lines
- Barcode and QR validation systems
- Machine vision inspection applications
- OCR and object detection workflows
- Shopfloor dashboards and analytics
- Embedded systems using Raspberry Pi and Arduino
- Custom desktop applications using Python and PySide6

I enjoy building systems that are not only functional, but also reliable, fast, and easy for operators to use on the shopfloor.

## 🚀 What I Build

### 🏭 Industrial Traceability Systems

I have worked on traceability systems for manufacturing lines where each station is connected with PLCs, barcode scanners, databases, and dashboards.

One major project included:

- 10 station interlocking system
- Barcode validation at every station
- Python + MySQL backend
- PLC-to-PLC and PLC-to-software communication
- Streamlit and Plotly dashboard
- SCADA feedback optimized to under 200 ms
